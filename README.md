## Overview

**Wellness Scape UAE** is a modern, high-performance wellness and spa website designed to provide an immersive, luxurious experience for visitors seeking holistic wellness services in the UAE. The website features a clean, minimal design with premium aesthetics, smooth animations, and a focus on conversion optimization.

### 🎯 Key Objectives

- **Premium Brand Positioning** - Elegant design that reflects luxury wellness
- **High Conversion** - Optimized booking flow and clear CTAs
- **Exceptional UX** - Smooth navigation, fast loading, intuitive interface
- **Mobile First** - Seamless experience across all devices
- **Performance Excellence** - 90+ Lighthouse score

## ✨ Features

### Core Features

| Feature | Description |
|---------|-------------|
| 🏠 **Home Page** | Hero section with video/gradient overlay, services grid, testimonials, gallery, booking form |
| 📄 **About Page** | Split layout with brand story, mission, vision, and key highlights |
| 💆 **Services Page** | Filterable service catalog with detailed descriptions and pricing |
| 💎 **Packages Page** | Tiered pricing cards with popular package highlight |
| 📞 **Contact Page** | Contact form with WhatsApp integration and Google Maps embed |
| 📱 **Responsive** | Fully responsive design optimized for all screen sizes |

| Feature | Implementation |

## 🎨 Design System

### Color Palette

| Color | Hex | Usage |
|-------|-----|-------|
| **Primary** | `#E8D8C3` | Soft Beige - Backgrounds, accents |
| **Secondary** | `#1F3D2B` | Dark Green - Text, footer, overlays |
| **Accent** | `#C8A96A` | Gold - CTAs, highlights, icons |
| **Background** | `#FAF9F7` | Off-white - Main background |
| **Text** | `#2B2B2B` | Dark gray - Body text |

### Typography

- **Font Family:** Inter (system fallback)
- **Headings:** Light weight (300-400) for elegance
- **Body:** Regular weight (400) for readability
- **Hierarchy:** Clear visual distinction between heading levels

### Spacing & Layout

- **Container:** Max-width 7xl (1280px)
- **Grid:** Responsive columns (1, 2, 3, 4)
- **Border Radius:** Rounded-2xl (16px) to rounded-3xl (24px)
- **Shadows:** Premium shadow system with hover states
